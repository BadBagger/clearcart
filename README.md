# ClearCart

ClearCart is a planned SoftSmith Android app.

## Repository Status

This repository is initialized as a placeholder so the app can be connected to
SoftSmith DevHub and expanded by future Codex chats.

## Publishing Rule

DevHub detects Android updates from GitHub Releases with APK assets attached.
Pushing source code alone is not enough.

When this becomes an Android app, publish updates by:

1. Bumping `versionCode`.
2. Setting a new `versionName`.
3. Running tests.
4. Building the APK.
5. Committing and pushing source.
6. Creating a GitHub Release with an APK attached.
